****🧠 Alzheimer Disease Prediction System**


**Overview****

This project is a machine learning-based system designed to predict the likelihood of Alzheimer’s Disease (AD) using patient data such as age, gender, education, and genetic factors (APOE4).

The system combines:

A Python Flask backend for machine learning predictions
A Random Forest Classifier trained on patient datasets
(Optional) A Java-based frontend application that interacts with the backend

The application provides real-time predictions along with a confidence score, helping support early detection and risk assessment.

**Features**
Machine learning-based Alzheimer’s prediction
Random Forest Classifier for robust accuracy
REST API built with Flask
Real-time prediction results
Confidence score output
Clean data preprocessing pipeline
Modular project structure
